To open console in Google Chrome

Keyboard Shortcuts:

  • Windows/Linux: Press Control + Shift + J
  • Mac: Press Command + Option + J

Steps to open the console:

  1. open the Chrome Menu in the upper-right-hand corner of the browser window
  2. select More Tools > Developer Tools

The console will either open up within your existing Chrome window, or in a new window. You may have to select the Console tab.
